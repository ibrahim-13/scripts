# Using Skills

## Claude CLI

To add skills to Claude Code (the CLI), you must place a **SKILL.md** file into the `$HOME/.claude/skills/SKILL_NAME/` directory for personal use across all projects, or `PROJECT_DIR/.claude/skills/SKILL_NAME/` for project-specific use.

- Directory for user skills: `$HOME/.claude/skills/SKILL_NAME`
- Directory for project specific skills: `PROJECT_DIR/.claude/skills/SKILL_NAME`
